# react-native-zxing-scanner

A zxing barcode scanner for React Native.

## Installation

```sh
npm install react-native-zxing-scanner
```

## Usage

```js
import ZxingScanner from "react-native-zxing-scanner";

// ...

const result = await ZxingScanner.multiply(3, 7);
```
